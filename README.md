### INSTALACIÓN:

```
* apt update && apt upgrade -y

* termux-setup-storage

* pkg install -y git

* git clone https://github.com/Scorpio28-Oficial/Osint

* cd Osint

* chmod 777 Osint.sh

* ./Osint.sh
```
